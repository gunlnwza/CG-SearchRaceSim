# Search Race

## Goal

Given the car initial state, it must drive through the `N` checkpoints `L` times.

## Format

- First line is the car state `x y vx vy angle`.
- Second line is the number of checkpoints `N`, and laps `L`
- `N` lines after are the checkpoints' position.

```
x y vx vy angle
N L
x_1 y_1
x_2 y_2
...
x_n y_n
```
